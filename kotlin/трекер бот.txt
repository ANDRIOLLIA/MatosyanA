Телеграм бот для слежки за динамикой цен скинов (CS2)

Заказчик(умный):
1. Добавлять и удалять отслеживаемый предмет (Пользователь отправляет ссылку на желаемый предмет, а потом, с помощью парсинга обновляет данные о цене)
2. Ежесекундное обновление данных
3. Оповещать, когда предмет достигнет указанной цены

	       -БОТ-
		 L Предметы
		    L Предмет
			L Название
			L Цена

Команды: 
1. Добавить предмет
2. Удалить предмет
3. Редактировать предмет
4. Получить все предметы


ДОБАВИТЬ ПРЕДМЕТ: 

1. УЗНАТЬ название
2. УЗНАТЬ ссылка на предмет
3. ДЛЯ КАЖДЫЙ предмет ИЗ список предметов ПОВТОРЯТЬ:
3.1. ПОЛУЧИТЬ название ИЗ предмет КАК название_1
3.2. ЕСЛИ название РАВЕН название_1 ТОГДА:
3.2.1. СОЗДАТЬ строка ИЗ название КАК ошибка
3.2.2. ПОКАЗАТЬ ошибка
3.3. ИНАЧЕ:
3.3.1. ПОЛУЧИТЬ цена ИЗ ссылка на предмет
3.3.2. СОЗДАТЬ предмет ИЗ название И ссылка на предмет
3.3.3. ДОБАВИТЬ предмет В список предметов


УДАЛИТЬ ПРЕДМЕТ:

1. УЗНАТЬ название
2. ДЛЯ КАЖДЫЙ предмет ИЗ список предметов ПОВТОРЯТЬ:
2.1. ПОЛУЧИТЬ название ИЗ предмет КАК название_1
2.2. СРАВНИТЬ название И название_1
2.3. ЕСЛИ название И название_1 РАВНЫ:
2.3.1. УДАЛИТЬ предмет
2.4. ИНАЧЕ ЕСЛИ название И название_1 НЕ РАВНЫ:
2.4.1. СОЗДАТЬ строка ИЗ название КАК ошибка
2.4.2. ПОКАЗАТЬ ошибка


РЕДАКТИРОВАТЬ ПРЕДМЕТ:

1. УЗНАТЬ индекс
2. ПОЛУЧИТЬ предмет ИЗ список предметов ГДЕ индекс РАВЕН индекс
3. УЗНАТЬ название
4. УЗНАТЬ ссылка на предмет
5. ДЛЯ КАЖДЫЙ предмет ИЗ список предметов ПОВТОРЯТЬ:
5.1. ПОЛУЧИТЬ название ИЗ предмет КАК название_1
5.2. ПОЛУЧИТЬ ссылка на предмет ИЗ предмет КАК ссылка на предмет_1
5.3 ЕСЛИ название РАВНО название_1 И ссылка на предмет РАВНО ссылка на предмет_1:
5.3.1. ЗАМЕНИТЬ название НА название_1 И ссылка на предмет НА ссылка на предмет_1
5.4. ИНАЧЕ:
5.4.1. СОЗДАТЬ строка ИЗ название И ссылка на предмет КАК ошибка
5.4.2. ПОКАЗАТЬ ошибка

ПОЛУЧИТЬ ВСЕ ПРЕДМЕТЫ:

1. ДЛЯ КАЖДЫЙ предмет ИЗ список предметов ПОВТОРЯТЬ:
1.1. ПОЛУЧИТЬ название ИЗ предмет КАК название
1.2. ПОЛУЧИТЬ цена ИЗ предмет КАК цена
1.3. СОЗДАТЬ строка ИЗ название И цена КАК вывод
1.4. ПОКАЗАТЬ вывод
