Стандартные типы данных. Назначение, принцип работы, использование

int(32 бит), byte(8 бит), short(16 бит), long(64 бит) - целочисленные типы данных, диапазон значений которых зависит от количества бит(от размера)
Применяется для работы, где не нужна дробная часть, например для индексов или счетчиков

float(32 бит), double(64 бит) - числа с плавающей точкой(дробные), количество цифр после точки зависит от количества бит(от размера)
Применяется для работы, где нужна точность, например с финансами

bool(8 бит) - хранит одно из значений true/false
Применяется в логическх выражениях, например для хранения состояния

char(16 бит) - хранит символ в формате Unicode
Применяется для работы с текстами, например для парсинга

String - неизменяемый объект, который хранит набор символов
Применяется для хранения текстовых данных, например для сообщений или ввода текста

Array - хранит коллекцию данных одного типа
Применяется для хранения коллекций однотипных данных, например для списка чисел
